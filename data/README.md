Simpan file GeoJSON hasil ekspor QGIS di folder ini jika ingin memuat data otomatis saat project dijalankan lewat HTTP.

Nama file yang didukung:
- `noise_data.geojson` untuk layer titik ukur dengan atribut `noise_db`
- `noise_measured.geojson` untuk layer titik ukur dengan atribut `noise_db`
- `study_area.geojson` untuk layer batas area opsional

Jika kamu membuka `index.html` langsung dari file browser, gunakan upload manual dari panel aplikasi.
